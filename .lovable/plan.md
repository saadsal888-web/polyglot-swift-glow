## التغييرات التفصيلية

### 1. `src/pages/Index.tsx` - إعادة بناء كاملة

**Header جديد** (مطابق للمرجع):
- يسار: `❤️ 5` + `💎 120` (نقاط)
- وسط: `WORDCARDS` + `تعلم الإنجليزية`
- يمين: أيقونة الحساب (دائرة)

**رسالة ترحيب**:
- `أنت في المكان المناسب — ابدأ عندما تكون مستعداً.`

**Hero Card - رحلتك التعليمية**:
- خلفية gradient بنفسجية
- زر `▶️` يسار + زر `ابدأ الدرس التالي` يمين
- عنوان: `رحلتك التعليمية`
- وصف: `تابع من حيث توقفت • 60 درس بانتظارك`
- شريط تقدم مع نسبة الإتمام
- عند الضغط → `/words` (الكلمات)

**بطاقة المحادثات اليومية**:
- أيقونة محادثة + عنوان
- قائمة المواضيع: السوق، الشارع، الصيدلية، الكوفي، المطار
- زرين: `مواقف حقيقية` + `تدريب مكثف`
- عند الضغط → `/phrases`

**بطاقة لوحة الصدارة**:
- أيقونة كأس + عنوان
- شارات المستويات: ماسي، ذهبي، فضي، برونزي
- عند الضغط → `/achievements`

**قسم أدوات التعلم** (شبكة 2x4):

| الأداة | الأيقونة | اللون | الوجهة |
|--------|----------|-------|--------|
| القواعد | 📖 | برتقالي | `/train-phrases` |
| الكلمات | 🧠 | بنفسجي | `/words` |
| التدريب | ⚡ | أصفر | `/spelling-practice` |
| القصص | 📚 | وردي | `/flashcards` |
| الأخطاء | ⭐ | وردي | `/difficult-words` |
| التوصيل | 🧩 | أخضر | `/exercise` |
| الصدارة | 🏆 | ذهبي | `/achievements` |
| قاموسي | ➕ | أزرق | `/library` |

### 2. التنسيق البصري

| العنصر | التفاصيل |
|--------|---------|
| Hero Card | `bg-gradient-to-br from-wc-purple to-indigo-600` + `rounded-3xl` + ظل |
| بطاقات المحادثات/الصدارة | `bg-white rounded-2xl border border-gray-100 shadow-sm` |
| أدوات التعلم | `bg-white rounded-2xl p-6` مع أيقونات دائرية ملونة |
| الأنيميشن | تأخيرات سريعة 0.05-0.15s |

### 3. إزالة العناصر القديمة

- بطاقة Streak
- بطاقة Progress Path مع Stats Grid
- قسم المسار التعليمي القديم
- Feature Cards المنفردة (كتابة، أخطاء، قاموسي، بطاقات)
- بانر التجربة المجانية (يبقى TrialTimer العائم)
