

# خطة: تحسين تجربة الصفحة الرئيسية

## التغييرات المطلوبة

### 1. رفع "الكلمات" فوق مباشرة بعد الترحيب

```text
قبل:                              بعد:
┌─────────────────────┐          ┌─────────────────────┐
│ Header              │          │ Header              │
├─────────────────────┤          ├─────────────────────┤
│ مرحباً بك 👋        │          │ مرحباً بك 👋        │
├─────────────────────┤          ├─────────────────────┤
│ تجربة مجانية        │          │ 🧠 الكلمات (كبير)   │ ← أول شيء يراه
├─────────────────────┤          ├─────────────────────┤
│ Streak Card         │          │ تجربة + Streak     │ ← صف واحد
├─────────────────────┤          ├─────────────────────┤
│ Progress Card       │          │ Progress (مصغر)    │
├─────────────────────┤          ├─────────────────────┤
│ المسار التعليمي     │          │ التحديات (2x2)     │
│ - الكلمات          │          ├─────────────────────┤
├─────────────────────┤          │ البطاقات السريعة    │
│ التحديات (2x2)     │          └─────────────────────┘
```

### 2. تصغير العناصر وضغطها

| العنصر | الحالي | الجديد |
|--------|--------|--------|
| Streak Card | `p-4` | `p-2.5` + مع التجربة في صف واحد |
| Progress Card | `p-5` + Stats Grid | `p-3` + إخفاء Grid |
| Feature Cards | `p-4` + كل واحد بسطر | `p-3` + Grid 2x2 |
| التأخيرات | 0.1, 0.15, 0.2... | 0.05, 0.08, 0.1... |

### 3. تسريع الأنيميشن

```typescript
// قبل
transition={{ delay: 0.5 }}

// بعد
transition={{ delay: 0.1, duration: 0.2 }}
```

### 4. فتح كل شيء للمشتركين

الـ `isPremium` موجود بالفعل في السياق ويُستخدم لإخفاء القفل. فقط نتأكد أن:
- جميع المستويات في `/words` تُفتح
- جميع الميزات تظهر بدون `Premium` badge
- `Hearts = ∞` للمشتركين

## الملفات المتأثرة

| الملف | التغيير |
|-------|---------|
| `src/pages/Index.tsx` | إعادة ترتيب + تصغير + تسريع |
| `src/pages/Words.tsx` | التحقق من `isPremium` لفتح الكل |
