
# نظام التجربة البسيط: 30 دقيقة + عرض محدود

## الفكرة

```text
┌─────────────────────────────────────────┐
│  🎁 30 دقيقة مجانية                      │
│         ↓                               │
│  🔥 "لحق عليه! عرض لفترة محدودة"         │
│     ينتهي خلال: 23:59:59                │
└─────────────────────────────────────────┘
```

## التغييرات

### 1. `src/hooks/usePremiumGate.ts`

| الحالي | الجديد |
|--------|--------|
| `TRIAL_DURATION = 86400` (24 ساعة) | `TRIAL_DURATION = 1800` (30 دقيقة) |

**إضافات:**
- `OFFER_DURATION = 86400` (24 ساعة للعرض)
- `offerTimeLeft` - الوقت المتبقي للعرض
- `isOfferActive` - هل العرض ساري

### 2. `src/components/subscription/TimeUpOverlay.tsx`

**التصميم الجديد:**
```text
┌──────────────────────────────────────────┐
│                   🔥                     │
│                                          │
│         لحق عليه!                        │
│      عرض لفترة محدودة                    │
│                                          │
│  ┌──────────────────────────────────┐   │
│  │  ⏰ ينتهي خلال:                   │   │
│  │      23 : 45 : 30                │   │
│  └──────────────────────────────────┘   │
│                                          │
│  ┌──────────────────────────────────┐   │
│  │    ١٢٠ ر.س  →  ٦٠ ر.س           │   │

│                                          │

```

### 3. `src/components/subscription/TrialTimer.tsx`

**أثناء الـ 30 دقيقة:**
```
🎁 تجربة مجانية: 29:45
```

**بعدها (العرض المحدود):**
```
🔥 لحق عليه: 23:45:30
```

## ملخص الملفات

| الملف | التغيير |
|-------|---------|
| `src/hooks/usePremiumGate.ts` | 30 دقيقة + عداد العرض 24 ساعة |
| `src/components/subscription/TimeUpOverlay.tsx` | "لحق عليه! عرض لفترة محدودة" + عداد |
| `src/components/subscription/TrialTimer.tsx` | عرض الدقائق ثم عداد العرض |
